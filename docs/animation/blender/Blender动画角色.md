# 动画

## 动画制作

- 物体约束

- 骨骼绑定

- 通过 **形态键** 变形

- 借助 mixamo
  - 导出模型 fbx 文件
    - 包括 limit to 选定的物体
    - 取消 烘培动画

  - 登录 mixamo
    - 上传 fbx 角色
    - 绑定基本位置，自动骨骼绑定
    - 选择动画
    - 导出绑定动画fbx 文件

  - 导入模型 fbx 文件






## 注意

- 模型的修改器，骨架要在细分前边，这样更加光滑自然
