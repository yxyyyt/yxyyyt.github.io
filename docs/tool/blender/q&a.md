- 正交视图和透视视图的区别
  
  正交视图和透视视图的区别在于它们呈现对象的方式不同。 正交视图忽略了远近关系，将所有物体都展现在一个平面上，使物体的大小和形状不会因角度或距离的变化而改变。 透视视图则在图像上模拟了透视效果，物体离观察者越远，则看起来越小，远离视线的物体看起来比靠近视线的物体小，这种效果使图像更具深度感和空间感。
